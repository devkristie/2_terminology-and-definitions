9. Explain the difference between prefixing and post-fixing increment/decrement operators.
If you use the ++ operator as a prefix like: ++var, the value of var is incremented by 1; then it returns the value.
If you use the ++ operator as a postfix like: var++, the original value of var is returned first; then var is incremented by 1.

10. What is operator precedence and how is it handled in JS?
The precedence of an operator specifies how "tightly" it binds two expressions together. For example, in the expression 1 + 5 * 3, the answer is 16 and not 18 because the multiplication ("*") operator has a higher precedence than the addition ("+") operator. Parentheses may be used to force precedence, if necessary. For instance: (1 + 5) * 3 evaluates to 18.

13. What are the eight data types in JavaScript?
String
String Data type is used to represent textual data. It is a set of "elements" of 16-bit unsigned integer values.

Boolean
Boolean is a primitive data type in javascript. Boolean data types consist of logical They consist of two possible values “true” or “false”.
BigInt
BigInt data type is for integer numbers of arbitrary length. With BigInt, you can safely store and operate on large integers even beyond the safe integer limit for numbers.
Null
The null data type represents the value 0 it means this data type did not consist any data.
Undefined
A variable that has not been assigned a value has the value undefined.
Symbol
The symbol data type is used for giving unique identifiers.
Object
In JavaScript, objects can be seen as a collection of properties. With the object literal syntax, a limited set of properties are initialized; then properties can be added and removed. Property values can be values of any type, including other objects, which enables building complex data structures. Properties are identified using key values. A key value is either a String or a Symbol value.

21. What is the difference between the slice/substring/substr string methods?
slice()
The slice() method extracts a part of a string.
The slice() method returns the extracted part in a new string.
The slice() method does not change the original string.
substring()
The substring() method extracts characters, between two indices (positions), from a string, and returns the substring.
The substring() method extracts characters from start to end (exclusive).
The substring() method does not change the original string.
substr()
The substr() method extracts a part of a string.
The substr() method begins at a specified position, and returns a specified number of characters.
The substr() method does not change the original string.

22. What are the three logical operators and what do they stand for?
|| (OR)
The “OR” operator is represented with two vertical line symbols:
result = a || b;
finds the first truthy value
&& (AND)
The AND operator is represented with two ampersands &&:
result = a && b;
finds the first falsy value
! (NOT)
The boolean NOT operator is represented with an exclamation sign !.

The precedence of NOT ! is the highest of all logical operators, so it always executes first, before && or ||.

34. What is function scope?
The variables which are declared inside the function body. That is a local function or function-level variable. We can not use them outside the function block.

35. What are return values?
JavaScript passes a value from a function back to the code that called it by using the return statement. The value to be returned is specified in the return. That value can be a constant value, a variable, or a calculation where the result of the calculation is returned.

36. What are arrow functions?
A shorter notation for writing functions.
let sum = (a, b) => a + b;
This is a shorter form of:
let sum = function(a, b) {
return a + b;
};
